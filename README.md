FULL STACK MERN PROJECT

Taskmanager web application using:
-backend: Node.js, MongoDB (Authentification by Log in, Sign up Functional, save the personal accoount Task instead of using LocalStorage)
-frontend: React, Javascript (UI/UX)

The project is very similar, it seems like to-do List but upgraded with database. User can now create there own Account, update the date, time and status for personal tasks. 

sign up for an account and add your daily tasks

PULL AND RUN 

-clone the project
-cd backend, npm install, npm start
-cd frontend, npm install, npm run dev
